# Teacher Evidence Index

This page maps the submission to likely automated and human assessment checks. Every linked artifact is either generated by a real project command or explicitly labeled as planned/not implemented.

## Reproduce the evidence

Run from the repository root on Windows/PowerShell:

```powershell
uv sync --extra dev
uv run cops-and-robbers play --config config/default_game.yaml --seed 42 --output results/report_email_preview.json
uv run python -m cops_and_robbers_rl.main mcp-smoke
powershell -File scripts/capture_cli_evidence.ps1
powershell -File scripts/capture_gui.ps1
uv run pytest --cov=cops_and_robbers_rl --cov-report=term-missing --cov-fail-under=85
uv run ruff check .
uv run ruff format --check .
```

Expected match output reports `sub_games: 6`, ordered games `1..6`, authoritative totals, `technical_failures_retried: 0`, and `email_mode: dry_run`. It creates ignored local previews at `results/report_email_preview.json` and `results/report_email_preview.txt`; the JSON uses anonymized placeholder identity unless private environment values are configured. Expected MCP output reports success, the dependency-safe in-process mode, different cop/thief ports, and legal structured actions. The current verified quality result is 49 passing tests and 89.26% coverage.
